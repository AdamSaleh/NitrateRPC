Allow upload of test results to TCMS directly from Jenkins.

Server URL
URL of TCMS server. Make sure you use HTTPS - otherwise access credentials would be sent to server unencrypted(also with Kerberos authentication).
Username and Password
Due to security concerns, these fields are not stored (they would be stored as plaintext in build config file). You will have to enter them again next time.
Path to test files
Path relative to each build, where TestNG .xml files with test results are stored.
